玉米鸡块

发表于 2016-11-07   |   分类于 烹饪食谱   |  

材料

* 250克鸡胸肉
* 150玉米饼
* 1个鸡蛋
* 2汤匙牛奶
* 65克面粉
* 1茶匙甜椒粉
* 1茶匙大蒜粉
* 1/4茶匙盐
* 1/4茶匙胡椒

步骤

1. 将玉米饼放到食品袋中用擀面杖敲碎，放入干净的碗中。
2. 在干净的碗中打入一个鸡蛋，放入牛奶，打散混匀。
3. 在干净的碗中放入面粉、辣椒粉，大蒜粉、盐和胡椒粉，混匀。
4. 鸡肉切小块，裹面粉混合物，沾一下鸡蛋混合液，再裹一层敲碎的玉米饼，放到烤纸上。
5. 入烤箱，200度烤15分钟。

npm常用命令

* npm init  会引导你创建一个package.json文件，包括名称、版本、作者这些信息等
* npm install <name> 安装nodejs的依赖包
* npm install <name> -g  将包安装到全局环境中
* npm install <name> --save  安装的同时，将信息写入package.json中。项目路径中如果有package.json文件时，直接使用* npm install方法就可以根据dependencies配置安装所有的依赖包
* npm remove <name> 移除
* npm update <name> 更新
* npm ls  列出当前安装的了所有包
* npm root  查看当前包的安装路径
* npm root -g  查看全局的包的安装路径
* npm help  帮助，如果要单独查看install命令的帮助，可以使用的npm help install

设计模式：Facade（外观）模式

发表于 2016-08-28   |   分类于 学习笔记   |  

结构型设计模式

Facade模式提供了一个方便的高层次接口，用来访问子系统中的一群接口，能够隐藏其底层的真实复杂性。Facade是一种结构型模式。
Facade模式既能简化类的接口，也能将这个类从使用它的代码中解耦。这使我们能够间接与子系统交互。
每当使用jQuery的$(el).css()或$(el).animate()方法时，实际上我们是在使用Facade:一种更简单的共有接口，使我们不必手动在jQuery核心中调用很多内部方法以便实现某些行为。
